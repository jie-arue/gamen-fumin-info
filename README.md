# gamen-fumin-info
# 画面不眠 (Gamennemin) - Android App

「画面不眠」は、スマートフォンの画面点灯状態を自由に制御し、指定した時間だけ消灯を防ぐことができるユーティリティアプリです。

## 概要
読書中、料理のレシピ確認中、デバッグ作業中など、「勝手に画面が消えてほしくない」シーンで活躍します。シンプルな操作で、必要な時だけ画面を明るく保ちます。

## 主な機能
- **画面点灯の維持**: アプリを有効にしている間、システムの自動スリープを無効化します。
- **明るさ調整**: 画面を暗く保ちつつ点灯させたい場合、明るさを20%〜100%の間で調整可能です。
- **復元機能**: 画面タップ時に、設定した明るさを一定時間（1分間）復元する機能を搭載しています。
- **自動終了タイマー**: 最大180分までのタイマーを設定でき、消し忘れを防止します。

## 動作環境
- Android OS 8.0以上推奨

## 導入・ビルド方法
本プロジェクトは Kotlin で開発されています。
1. `git clone https://github.com/jie-arue/gamen-fumin-info.git`
2. Android Studio でプロジェクトを開く
3. `local.properties` に必要なAPIキー等を設定（詳細はコード内のコメント参照）
